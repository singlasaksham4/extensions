AI Keyboard Shortcuts

How to Install and Use the AI Keyboard Shortcuts Extension

Installation Guide:

  1. 1.

    Download the extension's zip file:

    Download Zip File

    Make sure you know where this file is saved on your computer!

  2. 2.

    Extract the `.zip` file. You should get a folder containing the extension's files (like `manifest.json`, background scripts, etc.).

  3. 3.

    Open your Chrome browser and go to the Extensions page by typing:

    chrome://extensions/
  4. 4.

    Enable Developer Mode. Look for a toggle switch, usually in the top-right corner of the Extensions page, and turn it ON.

    Once Developer Mode is enabled, click on the "Load unpacked" button that appears (usually in the top-left).

    A file explorer window will pop up. Navigate to and select the folder you extracted in Step 2 (NOT the zip file itself).

  5. 5.

    Congratulations! You have successfully installed the AI Keyboard Shortcuts extension.

    Now you can use it! Open any website and you should be able to ask the extension for shortcuts. For example, try asking: "show me shortcuts for copy-pasting" or "how to undo a mistake?".

How to Use:

After installation, you can interact with the extension. Depending on how you've designed it, you might type a command into your browser's address bar (Omnibox) or a special input field if the extension provides one. Try asking for specific actions like "shortcuts for managing tabs" or "how to zoom in/out".
